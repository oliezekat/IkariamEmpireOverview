Ikariam Empire Overview (test)

Test case to use PHP built-in server to serve local static file as userscript under development.

  1. Disable any userscript which look like "Empire Overview",
  2. Log-in to any Ikariam server,
  3. Disable AdBlock browser add-on,
  4. Start your local web server (see README.md),
  5. Open "http://localhost/test" in any web browser,
  6. Install this userscript.
